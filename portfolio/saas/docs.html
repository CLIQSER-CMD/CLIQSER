Quickstart Guide

Welcome to the Neuravox API v4.2 reference. The Neuravox AI platform leverages secure REST endpoints and WebSockets to orchestrate distributed neural clusters, manage localized code bases, and query active task telemetry streams.

To get up and running:

  1. Log into your Command Console and generate an operations API key.
  2. Authenticate requests by including the token in your headers.
  3. Initialize your first neural agent using the endpoints below.

Authentication

Every REST API request directed to the Neuravox cluster must include your secret API key in the request headers under the name Authorization.

Example Header Structure
Authorization: Bearer nvx_live_51xLq8Fq89XyZ4T7bV90jK1
POST

/v4/agents/deploy

Deploys a custom neural agent node directly to your allocated workspace stack. Once compiled, this agent starts listening for pipeline triggers or API instructions.

Request Body Parameters

Parameter Type Description
name string The logical name of your new agent. Must be unique.
model string Framework code, e.g. engine-v4.2.
instructions string System prompt instructions guiding task behavior.
